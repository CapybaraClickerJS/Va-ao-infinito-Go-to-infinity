# Va-ao-infinito-Go-to-infinity
Vá ao Infinito: Desafie a gravidade neste arcade da Js Game Brazil! Suba plataformas, fuja da lava e chegue ao espaço. Inclui 6 níveis: do modo Bebê (Lava Arco-íris e imortal) ao Impossível. Use o sistema de magnetismo para saltos precisos. Simples de jogar, difícil de dominar. Até onde consegues subir? Criado por Js.
